
# Tourista

Explore the Rich Tapestry of Asia: Discover breathtaking landscapes, vibrant cultures, and timeless traditions on your next unforgettable journey.

## Live Site URL

  https://tourista-24a6c.web.app/


## Features of Tourista

- Comprehensive Destination Guides: Explore detailed insights into Asian destinations, from iconic landmarks to hidden gems.
- Personalized Trip Planning: Tailor your itinerary with customizable travel recommendations and insider tips.
- Cultural Experiences: Immerse yourself in authentic traditions through curated tours and local experiences.
- Secure Booking Platform: Ensure peace of mind with secure payment options and trusted accommodation partners.
- Engaging Content: Stay informed and inspired with articles, photos, and videos showcasing the beauty and diversity of Asia.

## Run Locally

Clone the project

```bash
  git clone https://github.com/programming-hero-web-course-4/b9a9-real-estate-JannatulAfrozFaria
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
